# 税务知识库

这是一个面向税务研究、职业学习和专业输出复用的 Obsidian 知识库。

## 从这里开始

- `purpose.md`：说明知识库目标、用途和范围。
- `schema.md`：说明 frontmatter 和内容规范。
- `index.md`：根目录导航页。
- `wiki/index.md`：知识层导航页。
- `CODEX.md`：Codex 维护本库时必须遵守的操作规范。

## 核心分层

- `inbox/`：临时收件箱，用来快速保存政策、案例、论文、网页和想法。
- `raw/`：原始资料层，是正式知识的 source of truth。
- `wiki/`：结构化知识层，从原始资料中整理出概念、规则、案例、政策和综合分析。
- `domains/`：学习路线和领域地图。
- `outputs/`：面试回答、税务 memo、研究报告、PPT、作品集材料。
- `templates/`：各类页面模板。
- `indexes/`：Dataview 查询和 MOC 导航页。
