# 美国销售税体系 (United States — Sales & Use Tax)

## 速览
- 无联邦层面VAT/GST，由各州独立征收Sales & Use Tax
- 仅在零售环节单次课征，无进项抵扣机制（与VAT本质不同）
- Wayfair案(2018)确立经济关联：远程卖家超$100K/200笔须征税


> 美国**没有联邦层面的 VAT/GST**，而是由州和地方政府征收零售销售税 (Sales Tax) 和使用税 (Use Tax)。2018年 Wayfair 案后，经济关联 (Economic Nexus) 成为核心概念。

## 一、速览

| 项目 | 内容 |
|------|------|
| **税名** | Sales and Use Tax |
| **联邦 VAT** | **无** |
| **征收层级** | 州 (State) + 地方 (County/City/District) |
| **有销售税的州** | **45 个州** + DC（5 个无销售税州：AK、DE、MT、NH、OR） |
| **税率范围** | 州税 0%–7.25%，加上地方税可达 **~11%** |
| **最高综合税率** | 路易斯安那部分地区约 **11.45%** |
| **申报周期** | 月度/季度/年度（因州而异） |
| **无统一税号** | 各州单独注册 |

## 二、核心概念：Nexus（关联性）

### 实体关联 (Physical Nexus)
- 在州内有办公室、仓库、员工、库存 → 有纳税义务

### 经济关联 (Economic Nexus) — Wayfair 案 (2018) ⭐
- 最高法院裁定：**无实体存在也可能有纳税义务**
- 典型门槛：年销售额 **USD 100,000** 或 **200 笔交易**（因州而异）
- 大多数州已采纳经济关联标准

### 点击关联 (Click-Through Nexus)
- 通过州内联盟营销商产生的销售超过门槛 → 视为有关联

## 三、征税范围

- **有形个人财产 (TPP)**：大多数州对 TPP 销售征税
- **服务**：各州差异极大 — 有的仅对少数服务征税，有的对大多数服务征税
- **数字产品**：越来越多州对数字产品征税（软件、流媒体、电子书）
- **SaaS/云服务**：约半数州将其视为应税

### 常见免税
- 食品杂货（多数州免税或低税率）
- 处方药
- 制造业原材料/设备（生产豁免）
- 转售 (Resale Exemption) — 持免税证明

## 四、Marketplace Facilitator 规则 ⭐

- 大多数州要求**市场促进者**（如 Amazon、eBay）代替第三方卖家收取和缴纳销售税
- 门槛通常与经济关联一致（USD 100,000/200 笔）
- 平台须在客户所在州注册并缴税

## 五、使用税 (Use Tax)

- 对未缴纳销售税的购买征收（如从免税州购买、网购未收税）
- 税率通常等同于当地销售税率
- 实际上对消费者的合规执行较弱

## 六、多州合规挑战

| 挑战 | 说明 |
|------|------|
| **税率碎片化** | ~13,000 个税务管辖区，税率各异 |
| **税基差异** | 各州对服务、数字产品征税范围不同 |
| **免税规则** | 各州免税清单、免税证明格式不同 |
| **申报频率** | 月/季/年因州和销售额而异 |
| **Streamlined Sales Tax (SST)** | 24 个州加入的自愿简化协议 |

## 七、申报与缴纳

- 各州独立注册、申报和缴纳
- 大多数州支持**电子申报**
- 部分州要求预缴 (Prepayment)
- 州税务机关可对 3–7 年内的申报进行审计

## 八、罚则

- 各州自行规定
- 常见模式：迟缴 5%–25% + 利息（年化 ~6%–12%）
- 欺诈可能面临刑事处罚

## 面试要点速记 🎯

1. **美国无联邦 VAT** — 与 EU/亚太体系的根本区别
2. **Wayfair 案 (2018)** — 经济关联革命，面试必考
3. **~13,000 个税务管辖区** — 合规复杂度极高
4. **5 个无销售税州** — AK, DE, MT, NH, OR
5. **Marketplace Facilitator** — 平台代收代缴趋势
6. **Use Tax** — 消费者对未缴销售税购买的自报义务
7. **税基不统一** — 服务和数字产品征税因州而异
8. **SST** — 简化协议但非强制
9. **无 input tax credit** — 销售税是单环节终端税，非增值税
10. **与 VAT 的本质区别** — 销售税仅对最终消费征税，无链式抵扣
