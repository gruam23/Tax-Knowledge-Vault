# 新加坡GST体系 (Singapore — Goods and Services Tax)

## 速览
- 单一税率9%（2024年1月起从8%上调），零税率适用出口
- 注册门槛SGD 1M年营业额，海外供应商注册制度(OVR)覆盖数字服务
- 季度申报，高度数字化的InvoiceNow电子发票推进中


> 新加坡 GST 以简洁高效著称，2024年1月起标准税率升至 9%。OVR 和 InvoiceNow 代表其数字化转型方向。

## 一、速览

| 项目 | 内容 |
|------|------|
| **税名** | Goods and Services Tax (GST) |
| **引入日期** | 1994年4月1日 |
| **管理机关** | Inland Revenue Authority of Singapore (IRAS) |
| **标准税率** | **9%**（2024年1月起，此前 8%） |
| **其他** | 零税率 (0%)、免税 |
| **税号格式** | M2/MR/19- + 7位 + 校验码；非居民 F2- |
| **申报周期** | 季度（可申请月度） |
| **注册门槛** | **SGD 1,000,000** |

## 二、税率

| 分类 | 税率 | 进项抵扣 |
|------|------|---------|
| **应税** | 9% | ✅ |
| **零税率** | 0% | ✅ |
| **免税** | 无 GST | ❌ |

- **零税率**：出口货物、国际服务
- **免税**：住宅不动产销售/租赁、特定金融交易、投资贵金属、数字支付令牌

## 三、纳税人

### 注册门槛
- **回溯**：年末应税营业额 > SGD 1M → 30 天内注册
- **前瞻**：预计未来 12 个月将超 SGD 1M → 30 天内注册
- 自愿注册须承诺至少 **2 年**

### GST 集团注册
- "共同控制"下的企业可组成 GST 集团
- 各成员须先单独注册，再合并为集团
- 集团内交易不征 GST
- 成员**连带责任**
- 非新加坡居民可加入（需满足条件）

### 非居民企业
- 在新加坡从事应税供应且超门槛 → 须注册
- 须指定**本地税务代理人** (Section 33(1) Agent)

## 四、反向征收

### 进口服务反向征收
- 从海外采购服务 + 采购方**无完全进项抵扣权** → 须反向征收
- 非注册人：采购海外服务 > SGD 1M/年 → 须按反向征收注册

### 低价值货物 (2023年1月起)
- ≤ SGD 400 的空运/邮递进口货物 → 纳入反向征收范围

### 国内反向征收 (Customer Accounting)
- 手机、存储卡、预装软件的 B2B 交易 ≥ SGD 10,000/单张发票
- 买方负责申报销项税

## 五、数字经济 — OVR 制度 ⭐

### 海外供应商注册 (Overseas Vendor Registration)
- 非居民向新加坡消费者 (B2C) 提供**数字服务** → 须注册
- 门槛：全球营业额 > SGD 1M 且新加坡 B2C 数字服务 > SGD 100,000
- **Pay-only 模式**：收取并缴纳 GST，但不可抵扣进项

### 2023年1月扩展
- OVR 扩展至 B2C **非数字远程服务**和**低价值货物**
- B2B 数字服务 → 买方反向征收（非注册供应商无须注册）

## 六、进项税抵扣

### 不可抵扣
- 私人用途支出
- 私人汽车（购买/租赁/维护/运营）
- 员工医疗保险
- 休闲俱乐部会费

### 按比例抵扣 (Partial Exemption)
- 两阶段法：直接归属 + 残余分摊
- **微量规则**：月均免税供应 ≤ SGD 40,000 且 ≤ 总供应 5% → 视为全额可抵

### 退款
- 进项 > 销项 → 差额退还（季度申报一般 3 个月内，月度 1 个月内）
- 逾期退款付息（按主要贷款利率）

## 七、InvoiceNow 电子发票 ⭐

新加坡推行 **GST InvoiceNow**：

| 日期 | 适用范围 |
|------|---------|
| **2025年11月** | 新成立的自愿注册公司 |
| **2026年4月** | 所有新自愿 GST 注册者 |
| **后续阶段** | 新强制注册者 + 现有注册者（待公布） |

- 通过 **InvoiceNow 网络**（基于 Peppol）传输发票数据至 IRAS
- OVR 和反向征收注册者暂免

## 八、特殊方案

| 方案 | 说明 |
|------|------|
| **MES（主要出口商）** | 进口 GST 延期，改善现金流 |
| **IGDS（进口延期缴纳）** | 进口 GST 延至月度申报时缴纳 |
| **ACMT** | 合同制造商/贸易商方案 |
| **现金制** | 营业额 < SGD 1M，按收付实现制 |

## 九、申报与缴纳

| 项目 | 规则 |
|------|------|
| **频率** | 季度（可申请月度） |
| **截止** | 期末后 **1 个月** |
| **电子申报** | **强制**（myTax Portal） |
| **缴纳方式** | GIRO、网银、PayNow |
| **年度申报** | 不要求 |

## 十、罚则

| 违规 | 罚则 |
|------|------|
| 迟注册 | SGD 10,000 + 税额 10% |
| 迟缴税款 | 5% + 60天后 2%/月（上限 50%） |
| 迟交申报 | SGD 200 + SGD 200/月（上限 SGD 10,000） |
| 错误 | 短缴税额 2 倍 + 最高 SGD 5,000 或 3 年监禁 |
| 欺诈 | 短缴税额 3 倍 + 最高 SGD 10,000 或 7 年监禁 |
| 反避税附加税 | 额外 GST 的 **50%** |
| 追诉时效 | **5 年**（欺诈无时限） |

## 面试要点速记 🎯

1. **9% 单一税率**（2024年起）— 简洁高效
2. **SGD 1M 注册门槛** — 相对较高
3. **OVR 制度** — 数字经济征税先进模式
4. **InvoiceNow** — 基于 Peppol 的电子发票（2025年起）
5. **反向征收**仅限无完全抵扣权者 — 全额抵扣企业免除
6. **低价值货物 SGD 400** — 2023 年起纳入
7. **Customer Accounting** — 国内反向征收（手机、存储卡）
8. **IGDS 延期缴纳** — 改善进口商现金流
9. **Partial exemption 微量规则** — SGD 40,000/月 + 5%
10. **欺诈无追诉时限** — 新加坡执法严格
