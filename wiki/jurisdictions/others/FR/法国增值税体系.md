# 法国增值税体系 (France — TVA)

## 速览
- 四档税率：20%标准/10%中间/5.5%减低/2.1%超减低
- 电子发票改革：2026年9月起大企业强制，2027年全面推行
- 非居民企业通常须指定税务代理(Fiscal Representative)


> 法国是 VAT 发源国（1954年），TVA 制度在 EU 框架内有许多独特规定，尤其在电子发票强制化方面走在前列。

## 一、速览

| 项目 | 内容 |
|------|------|
| **税名** | Taxe sur la Valeur Ajoutée (TVA) |
| **引入日期** | 1954年4月10日 |
| **标准税率** | **20%** |
| **减低税率** | **10%、5.5%、2.1%** |
| **税号格式** | FR + 2位校验码 + 9位 SIREN |
| **申报周期** | 月度/季度/年度 |
| **注册门槛** | 无（小企业豁免：货物 ≤ EUR 91,900；服务 ≤ EUR 36,800） |

## 二、税率

| 税率 | 适用范围 |
|------|---------|
| **20%** | 大多数货物和服务 |
| **10%** | 餐饮堂食、客运、住宿、家装、非报销药品 |
| **5.5%** | 基本食品、书籍、能源、卫生用品、社会住房 |
| **2.1%** | 报销处方药、特定出版物、现场演出前140场 |
| **0%** | 共同体内供应、出口 |
| **免税** | 金融/保险、医疗、教育、住宅租赁 |

**特殊**：DOM 海外领土标准税率 8.5%；法属波利尼西亚/新喀里多尼亚不适用法国 TVA。

## 三、纳税人

### VAT 集团 (Groupe TVA)
- 自 **2023年1月1日起**正式引入
- 成员须有财务、经济和组织紧密联系
- 集团内交易不征 TVA，代表成员负责申报
- 最低存续期 **3 年**，成员承担**连带责任**

### 非居民企业
- 非 EU 企业须指定**税务代理人** (Représentant fiscal)
- EU 企业无须代理人，直接登记

## 四、反向征收

- 跨境 B2B 服务 → 买方反向征收
- 国内反向征收：建筑分包、工业废料、CO₂ 配额、天然气/电力（经销商）

## 五、数字经济

- B2C 电子服务按消费者所在国征税（可用 OSS）
- 在线平台促成特定 B2C 交易时视为供应商

## 六、进项税抵扣

### 不可抵扣
- 员工住宿、旅客运输、私人用途支出

### 法国特色：两层抵扣系数
1. **归属系数**：经济活动 vs 非经济活动
2. **课税系数**：应税 vs 免税
- 最终抵扣比例 = 归属系数 × 课税系数 × 允许系数

### 资本品调整
- 动产 **5 年**；不动产 **20 年**（EU 内最长之一）

## 七、强制电子发票 ⭐

法国在 EU 中**率先强制推行 B2B 电子发票**：

| 日期 | 内容 |
|------|------|
| **2026年9月** | 大型企业发出 + 所有企业接收 |
| **2027年9月** | 中型企业发出 |
| **2028年9月** | 小微企业发出 |

- 平台：**Chorus Pro**（B2G）或认证 **PDP** 平台
- 格式：**Factur-X**、CII、UBL
- 同步实施 **e-reporting**：非 FR 交易和 B2C 数据上报

## 八、申报与缴纳

| 项目 | 规则 |
|------|------|
| **频率** | 一般月度；年度 TVA < EUR 4,000 可季度 |
| **截止** | 次月 19–24 日 |
| **电子申报** | 大多数企业强制 |
| **补充申报** | EMEBI（替代 DEB）、EU 销售清单 |

## 九、罚则

| 违规 | 罚则 |
|------|------|
| 迟交申报 | 10%（无催告）→ 40%（催告后） |
| 迟缴税款 | 0.2%/月利息 |
| 故意隐瞒 | 40%（欺诈 80%） |
| 发票缺陷 | 每项 EUR 15，上限发票金额 25% |
| 追诉时效 | 一般 3 年，欺诈 6 年 |

## 面试要点速记 🎯

1. **法国是 VAT 发源国**（1954年）
2. **四档税率** 20/10/5.5/2.1% — 与德国双税率对比
3. **Groupe TVA** 2023 年起 — 集团内免 TVA
4. **强制电子发票** 2026–2028 — EU 中进度最快
5. **非 EU 企业须指定税务代理人**
6. **两层抵扣系数** — 法国特色
7. **不动产 20 年调整期** — EU 最长之一
8. **DOM 海外领土**适用不同税率
9. **DEB 已被 EMEBI 替代**
10. **迟缴利率 0.2%/月** — 远低于德国 1%/月
