# 日本消费税体系 (Japan — Consumption Tax / 消費税)

## 速览
- 标准税率10%，减低税率8%（食品饮料/报纸）
- 2023年10月起实施合格发票制度(Invoice System)，未注册者无法开具抵扣凭证
- 跨境B2C数字服务由非居民企业注册申报，B2B适用反向征收


> 日本消费税 (JCT) 于 1989 年引入，2023 年 10 月实施合格发票制度 (Qualified Invoice System) 是近年最重大改革。

## 一、速览

| 项目 | 内容 |
|------|------|
| **税名** | 消费税 (Consumption Tax, CT / Shouhizei) |
| **引入日期** | 1989年4月1日 |
| **管理机关** | 国税厅 (NTA) |
| **标准税率** | **10%**（国税 7.8% + 地方税 2.2%） |
| **减低税率** | **8%**（国税 6.24% + 地方税 1.76%） |
| **税号格式** | T + 13位数字 |
| **申报周期** | 月度/季度/半年/年度 |
| **注册门槛** | **JPY 10,000,000**（基准期为前两个会计年度） |

## 二、税率

| 税率 | 适用范围 |
|------|---------|
| **10%** | 大多数货物和服务 |
| **8%** | 食品和饮料（不含酒精饮料和堂食）；定期刊物订阅（每周至少 2 期） |
| **0% (免税含抵扣)** | 出口货物/服务、国际运输、外交使团供应、免税店 |
| **免税 (无抵扣)** | 银行利息、保险、教育、土地销售/租赁、社会福利 |

## 三、纳税人

### 小企业免税
- **基准期**营业额 ≤ JPY 10M → 免税
- 基准期 = 当前会计年度前两个年度
- **特定期间**（前一年度的前 6 个月）营业额也须 ≤ JPY 10M
- 新设立公司：注册资本 ≥ JPY 10M → 自动成为纳税人
- 2024年10月起：外国企业不再能用工资替代营业额标准

### 合格发票制度 (Qualified Invoice System, 2023年10月起) ⭐
- 须注册为**合格发票发行者**才能开具合格发票
- 买方只能就**持有合格发票**的购买抵扣进项税
- 过渡期：非合格发票可抵 80%（至 2026.9）→ 50%（至 2029.9）→ 0%

### 集团注册
- **不允许**集团消费税注册

### 非居民企业
- 成为纳税人后须指定**居民税务代理人**

## 四、反向征收

- **B2B 数字服务**：外国企业向日本企业提供 → 买方反向征收
- **演艺/运动员服务**：外国演艺/运动员在日本的服务
- 例外：买方应税销售比例 ≥ 95% 或使用简化制 → 无须反向征收

### B2C 数字服务
- 非居民须在日本注册、收取消费税并申报
- 除非可享受小企业免税

## 五、平台经济 (2025年4月起) ⭐

- **指定平台运营商**制度：NTA 指定的大型平台（年数字服务 > JPY 5B）
- 通过指定平台的外国 B2C 数字服务 → 视为平台提供，平台代缴消费税
- 未通过指定平台的外国 B2C → 外国供应商自行注册缴税

## 六、进项税抵扣

### 基本规则
- 须持有**合格发票**（2023年10月起）
- 用于经营目的方可抵扣
- 时限：**5 年**内申请

### 按比例抵扣
- 应税销售比例 < 95% 或营业额 > JPY 500M → 须按比例抵扣
- 两种方法：
  - **比例法**：进项税 × 应税销售比例
  - **逐项法**：直接归属应税/非应税，共用部分按比例

### 简化制 (Simplified System)
- 年营业额 ≤ JPY 50M → 按行业固定购入比率（40%–90%）计算抵扣
- 2024年10月起：无常设机构的外国企业不再适用

## 七、发票

### 合格发票必要内容
- 合格发票发行者的名称和注册号
- 交易日期
- 货物/服务描述
- 按税率分类的含税/不含税金额
- 适用税率和消费税额

### 简化合格发票
- 零售、餐饮、出租车等行业 → 可使用简化发票（无须注明买方名称）

### 电子发票
- 允许但非强制
- 须符合**电子记录保存法 (ERRL)** 要求

## 八、申报与缴纳

| 项目 | 规则 |
|------|------|
| **法人** | 会计年度结束后 **2 个月**内（可延长 1 个月） |
| **个人** | 次年 **3月31日** |
| **电子申报** | 资本金 > JPY 100M 的法人**强制** |
| **中间申报** | 国税 > JPY 480,000 → 半年；> JPY 4M → 季度；> JPY 48M → 月度 |

## 九、罚则

| 违规 | 罚则 |
|------|------|
| 迟缴税款 | 前 2 个月 7.3%/年 → 之后 14.6%/年（取较低的特别标准利率） |
| 迟交申报 | 自愿补报 5%；审计通知后 10%–30%（按金额阶梯） |
| 少报税额 | 自愿披露 0%；审计后 5%–15% |
| 欺诈 | 故意少报 35%；故意未申报 40%；可加重 10% |
| 公司高管责任 | 逃税 → 最高 **10 年**监禁或 JPY 10M 罚款 |
| 追诉时效 | **5 年**（欺诈 7 年） |

## 面试要点速记 🎯

1. **双税率 10%/8%** — 食品（不含酒精和堂食）适用 8%
2. **合格发票制度 (2023.10)** — 日本近年最大 VAT 改革，过渡到 2029 年
3. **基准期制度** — 前两年营业额决定当年纳税义务
4. **不允许集团注册** — 与 EU/澳大利亚不同
5. **指定平台制度 (2025.4)** — 大型平台代缴 B2C 数字服务消费税
6. **B2B 数字服务反向征收** — 但 95% 应税比例免除
7. **简化制 40%–90% 购入比率** — 小企业专用
8. **TOGC 不适用** — 业务转让照常征税
9. **5 年进项税申请时限** — 国际对比
10. **外国企业须指定居民税务代理人**
