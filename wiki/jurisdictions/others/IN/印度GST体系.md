# 印度GST体系 (India — Goods and Services Tax)

## 速览
- 2017年统一取代多种间接税，四档税率：5%/12%/18%/28%
- 双轨架构：邦内CGST+SGST，跨邦IGST，中央统一征管
- 电子发票(e-Invoice)和电子运单(e-Way Bill)强制推行


> 印度于 2017年7月实施统一 GST，采用**双重 GST 模式**（中央 + 州政府同时征收），是全球最复杂的间接税制度之一。

## 一、速览

| 项目 | 内容 |
|------|------|
| **税名** | Goods and Services Tax (GST) |
| **引入日期** | 2017年7月1日 |
| **结构** | 双重 GST：CGST（中央）+ SGST/UTGST（州/联邦领土）+ IGST（跨州） |
| **税率** | **0%、0.25%、3%、5%、18%、28%、40%** + 补偿附加税 |
| **税号格式** | GSTIN = 州代码(2位) + PAN(10位) + 实体号(2位) + 校验(1位) |
| **申报周期** | 月度/季度/年度 |
| **注册门槛** | 货物 INR 400万（部分州 INR 200万/100万）；服务 INR 200万 |

## 二、双重 GST 结构 ⭐

| 交易类型 | 征税 | 说明 |
|---------|------|------|
| **州内供应** | CGST + SGST | 各占一半（如 18% = 9% CGST + 9% SGST） |
| **跨州供应** | IGST | 等于 CGST + SGST 之和 |
| **进口** | IGST + 补偿附加税 | 作为关税一部分征收 |

### 进项抵扣规则
- IGST 抵扣优先用于 IGST → 可用于 CGST/SGST
- CGST 只能抵 CGST → 然后 IGST
- SGST 只能抵 SGST → 然后 IGST
- **CGST 不能抵 SGST，反之亦然**

## 三、税率

| 税率 | 适用范围 |
|------|---------|
| **0%** | 出口、经济特区供应 |
| **0.25%/3%** | 宝石、黄金、白银 |
| **5%** | 煤炭、经济舱机票、餐饮、住宅建设 |
| **18%** | 大多数服务、电器、商务舱机票、银行服务 |
| **28%** | 烟草制品 |
| **40%** | SUV、碳酸饮料、赛马/赌场、在线博彩 |
| **免税** | 水果蔬菜、通行费、TOGC |

**补偿附加税 (Compensation Cess)**：仅对烟草和烟草制品征收。

## 四、反向征收

### 进口服务
- 从印度境外接收服务 → 印度接收方**反向征收 IGST**
- 接收方须开具自制发票 (self-bill)

### 国内反向征收
- 适用于特定供应：货运代理、律师服务、赞助服务、汽车租赁
- 从未注册人处采购特定服务 → 注册接收方反向征收

## 五、数字经济

### OIDAR 服务（在线信息和数据库访问/检索）
- 境外供应商向印度未注册消费者 (B2C) → 须注册并缴 GST
- B2B → 印度注册接收方反向征收

### 在线博彩
- 境外供应商须注册并缴 GST，无论接收方是否注册

### 电商平台
- 须在源头代扣税款 (TCS)：应税供应价值的 **1%**
- 特定服务（住宿、客运、餐饮、本地配送）→ 平台视为供应商

## 六、进项税抵扣

### 不可抵扣
- 载客机动车（≤ 13座，特定情况除外）
- CSR 活动支出
- 健身/俱乐部会员费
- 不动产建设（自用，非转售）
- 个人消费
- 赠品/免费样品

### 发票匹配制度
- 进项抵扣须与供应商上报的发票**匹配**
- 供应商未在其申报中报告 → 买方不可抵扣

### 180天付款规则
- 开票 180 天内未付款 → 已抵扣的进项须**转回**

## 七、电子发票 ⭐

- 年营业额 > INR 5,000万 → **B2B 强制电子发票**
- 须通过 GST 电子门户获取 **IRN（发票参考号）**
- 年营业额 > INR 50亿 → B2C 须提供**动态二维码**
- B2C 电子发票试点已启动

## 八、申报与缴纳

| 项目 | 规则 |
|------|------|
| **GSTR-1（销项）** | 每月 **11日**前 |
| **GSTR-3B（汇总）** | 每月 **20日**前 |
| **季度申报** | 营业额 < INR 5,000万可选 |
| **年度申报** | 12月31日前（营业额 ≤ INR 2,000万可免） |
| **组合方案** | 年度申报 + 季度缴纳（制造商/贸易商 1%，餐饮 5%） |
| **迟缴利率** | **18%/年** |

## 九、罚则

| 违规 | 罚则 |
|------|------|
| 未注册 | INR 20,000 |
| 未/少缴税 | INR 20,000 或税额 10%（取较高者） |
| 欺诈 | INR 20,000 或税额 100%（取较高者） |
| 迟交申报 | 每日罚款，上限 INR 10,000 |
| 追诉时效 | 非欺诈 42 个月；欺诈亦 42 个月（2024-25 起统一） |
| 私人公司董事 | 可追究**连带责任** |

## 面试要点速记 🎯

1. **双重 GST** — CGST + SGST/IGST 结构，全球独特
2. **CGST 不能抵 SGST** — 进项抵扣交叉规则
3. **7 档税率**（0%~40%）— 全球最复杂税率体系之一
4. **发票匹配制度** — 供应商未报告则买方不可抵扣
5. **180天付款规则** — 未付款须转回进项
6. **电商 TCS 1%** — 平台代扣税款
7. **OIDAR 境外供应商须注册** — B2C 数字服务
8. **强制电子发票** INR 5,000万门槛
9. **组合方案** — 小企业简化（1%/5% 统一税率）
10. **补偿附加税仅对烟草** — 设计目的是弥补各州税收损失
