# 加拿大GST/HST体系 (Canada — GST / HST / QST / PST)

## 速览
- 联邦GST 5% + 省级PST/QST各不相同，部分省合并为HST(13-15%)
- 小企业免税门槛CAD 30,000，季度/年度申报可选
- 数字产品和服务：非居民年销CAD 30,000+须简化注册


> 加拿大的间接税高度碎片化：联邦 GST + 参与省 HST + 魁北克 QST + 三省独立 PST，形成全球独特的多层间接税体系。

## 一、速览

| 项目 | 内容 |
|------|------|
| **联邦税** | GST 5% |
| **引入日期** | 1991年1月1日（GST）；1997年4月1日（HST） |
| **管理机关** | Canada Revenue Agency (CRA)；魁北克由 Revenu Québec |
| **注册门槛** | **CAD 30,000**（公共服务机构 CAD 50,000） |
| **税号格式** | 15位（9数字 + 2字母 + 4数字） |
| **申报周期** | 月度/季度/年度 |

## 二、多层税率结构 ⭐

| 省/地区 | 税种 | 税率 |
|---------|------|------|
| **安大略** | HST | **13%** (5% + 8%) |
| **新不伦瑞克、纽芬兰、PEI** | HST | **15%** (5% + 10%) |
| **新斯科舍** | HST | **14%** (5% + 9%，2025年4月起) |
| **魁北克** | GST + QST | **14.975%** (5% + 9.975%) |
| **BC、马尼托巴** | GST + PST | 5% + **7%** |
| **萨斯喀彻温** | GST + PST | 5% + **6%** |
| **阿尔伯塔、三领地** | 仅 GST | **5%** |

- HST 省份：省税融入联邦 GST 体系，统一征收
- PST 省份：省税单独征收，规则各异（非增值税，多为零售税）
- 魁北克 QST：独立体系但基本与 GST 协调

## 三、纳税人

### 小供应商豁免
- 过去 4 个季度全球应税+零税率供应 < **CAD 30,000** → 免注册
- 单季度超过 CAD 30,000 → 该笔交易起即须注册
- 出租车/网约车**无豁免**

### 集团注册
- **不允许**集团 GST/HST 注册
- 但"密切关联"实体可选择将相互供应**视为无对价** (nil consideration election)
- 条件：90% 以上股权 + 90% 投票权 + 专事商业活动
- 联合选举方对应税义务承担**连带责任**

### 非居民企业
- 不在加拿大开展业务 → 可自愿注册以申请 ITC
- 无加拿大常设机构 → 须提供**保证金**（CAD 5,000 – CAD 1,000,000）
- **无须**指定税务代理人

## 四、反向征收 (Self-Assessment)

- 从未注册非居民进口**服务或无形资产** + 非 90% 以上用于商业活动 → 自我评估 GST
- **无国内反向征收**（但 2025 年预算提议电信行业试点 RCM）

## 五、数字经济 (2021年7月起) ⭐

### 非居民数字供应商
- 向加拿大消费者提供数字产品/服务 > CAD 30,000/12个月 → **须注册**
- 可使用**简化注册系统**：在线门户收缴，不可申请 ITC
- 分销平台运营商须代替非注册供应商收缴

### 仓储配送 (Fulfillment Warehouse)
- 平台销售加拿大仓库中的货物 → 须按正式 GST 规则注册
- 非居民供应商自有账号销售仓库货物 → 同样须注册

### 短租住宿平台
- 平台促成 > CAD 30,000 未注册房主的短租 → 平台视为供应商
- 非居民平台可选简化注册

### 魁北克 QST 指定注册
- 无实体存在的非居民在魁北克供应数字产品/服务 → 须向 Revenu Québec 注册
- 2019年起生效（早于联邦规则）

## 六、零税率和免税

### 零税率 (0%)
- 出口货物和服务
- 基本食品
- 国际运输
- 处方药
- 医疗器械
- 农业/渔业特定投入

### 免税供应
- 二手住宅
- 金融交易
- 慈善机构和公共部门多数供应
- 医疗保健、教育

## 七、进项税抵扣 (ITC)

### 不可抵扣
- 用于商业活动 < 10% 的购买
- 社交俱乐部会费
- 商务餐饮招待费的 **50%**
- 员工礼品（有例外）

### 按比例抵扣
- 两阶段法：直接归属 + 合理方法分摊
- 须采用"公平合理且持续使用"的方法

### 时限
- 一般 **4 年**内申请
- 大企业（> CAD 6M 年应税供应）和金融机构 → **2 年**

## 八、申报与缴纳

| 项目 | 规则 |
|------|------|
| **频率** | 年供应 > CAD 6M → **月度**；CAD 1.5M–6M → **季度**；< CAD 1.5M → **年度** |
| **截止** | 月度/季度：期末后 **1 个月**；年度：3 个月 |
| **电子申报** | 多数注册人强制 |
| **缴纳方式** | 银行转账、在线支付 |

### 特殊方案
- **快速方法** (Quick Method)：年应税供应 ≤ CAD 400,000，按简化费率计算
- **简化 ITC 方法**：购买 ≤ CAD 30，无须持有发票

## 九、罚则

| 违规 | 罚则 |
|------|------|
| 迟交申报 | 1% 税额 + 0.25%/月（最多 12 个月） |
| 迟缴税款 | 规定利率复利（每季调整） |
| 重大过失 | 净税额 25% 或 CAD 250（取较高者） |
| 追诉时效 | 一般 **4 年** |

## 面试要点速记 🎯

1. **多层税制** — GST/HST/QST/PST 四种并存，全球独特
2. **CAD 30,000 小供应商门槛** — 全球最低之一
3. **不允许集团注册** — 但有 nil consideration election
4. **简化注册制** — 非居民数字供应商专用
5. **Fulfillment warehouse 规则** — 类似 EU 的平台经济
6. **魁北克先于联邦** — 2019 年即推行数字经济 QST
7. **50% 餐饮招待限制** — 与美国规则类似
8. **ITC 时限 4 年 / 大企业 2 年** — 注意差异
9. **HST 省份统一征收** vs PST 省份独立征收
10. **2025 年电信 RCM 提案** — 加拿大首次考虑国内反向征收
