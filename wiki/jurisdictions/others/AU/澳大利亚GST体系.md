# 澳大利亚GST体系 (Australia — Goods and Services Tax)

## 速览
- 单一税率10%，GST-free(0%)适用基本食品/医疗/教育/出口
- 注册门槛AUD 75,000，非居民无门槛
- BAS(Business Activity Statement)季度/月度申报，ABN体系统一管理


> 澳大利亚于 2000 年引入单一税率 GST，由联邦政府管理，收入拨归各州。数字经济规则（Netflix Tax）走在全球前列。

## 一、速览

| 项目 | 内容 |
|------|------|
| **税名** | Goods and Services Tax (GST) |
| **引入日期** | 2000年7月1日 |
| **管理机关** | Australian Taxation Office (ATO) |
| **标准税率** | **10%** |
| **其他** | 零税率 (GST-free)、进项税受限 (Input-taxed)、免税 |
| **税号格式** | ABN（Australian Business Number）11位 |
| **申报周期** | 月度/季度/年度 |
| **注册门槛** | **AUD 75,000**（非营利 AUD 150,000） |
| **非居民退税** | 有（Tourist Refund Scheme / 直接退税申请） |

## 二、税率与分类

| 分类 | 税率 | 进项抵扣 | 示例 |
|------|------|---------|------|
| **应税供应** | 10% | ✅ 可 | 大多数货物和服务 |
| **GST-free** | 0% | ✅ 可 | 基本食品、医疗、教育、出口、儿童保育 |
| **Input-taxed** | 无 GST | ❌ 不可 | 金融服务、住宅租赁/销售 |

**关键区分**：GST-free ≠ Input-taxed。前者有进项抵扣权，后者没有。

## 三、纳税人

### 注册门槛
- 年营业额 ≥ **AUD 75,000** → 强制注册
- 非营利组织 ≥ **AUD 150,000** → 强制注册
- 出租车/网约车运营商 → **无门槛**，须注册
- 自愿注册：无门槛限制

### GST 集团注册
- 90% 以上控制的实体可组成 GST 集团
- 集团内交易不征 GST
- 代表成员负责申报
- 成员承担**连带责任**

### 非居民企业
- 在澳供应应税货物/服务且超门槛 → 须注册
- 可选择**简化注册** (Simplified GST Registration) — 无须 ABN，不可抵扣进项

## 四、反向征收

- 从海外接收的 B2B 服务/无形资产 → 买方**反向征收**
- 条件：进口方无完全进项抵扣权时才须实际缴纳
- 与澳大利亚境内不动产相关的服务 → 非居民供应商须注册并收 GST

## 五、数字经济 (Netflix Tax) ⭐

自 2017年7月1日起：
- 非居民向澳大利亚消费者提供**数字产品和服务** → 须注册并收取 GST
- 门槛：全球营业额 ≥ AUD 75,000 且澳大利亚 B2C 数字供应 ≥ AUD 75,000
- 涵盖：流媒体、在线游戏、电子书、软件、在线课程
- **平台规则**：电子分销平台 (EDP) 视为供应商 — 须代收 GST

### 低价值进口货物 (2018年7月起)
- AUD 1,000 以下的进口货物 → 非居民供应商/平台须收取 GST
- 替代了原来海关免征的做法

## 六、进项税抵扣

### 不可抵扣
- Input-taxed 供应相关进项（金融服务、住宅租赁）
- 私人用途支出
- 娱乐费用（大部分）
- 汽车购买中超过奢侈品税门槛的部分

### 可抵扣
- 商务差旅、住宿
- 汽车（业务用途）
- 广告、办公用品

### 按比例抵扣
- 同时用于应税和 input-taxed → 按比例分摊
- **微量规则**：input-taxed 进项 ≤ 总进项的较小比例 → 可全额抵扣

### 资本品调整
- 重大购置 ≥ AUD 500,000 → **10 年**调整期（不动产）
- 其他资本品 → **5 年**调整期

## 七、申报与缴纳

| 项目 | 规则 |
|------|------|
| **频率** | 年营业额 ≥ AUD 20M → **月度**；其他 → **季度** |
| **截止** | 期末后 **21 天**（月度）或 **28 天**（季度） |
| **电子申报** | 强制（通过 ATO Business Portal 或税务代理） |
| **年度申报** | 无单独年度 GST 申报 |

## 八、罚则

| 违规 | 罚则 |
|------|------|
| 迟交申报 | **1 penalty unit/28天**（2026年 = AUD 330/单位） |
| 迟缴税款 | **8%/年** General Interest Charge (GIC) |
| 错误申报 | 25%–75%（视过错程度） |
| 欺诈 | 最高 75% 罚款 + 刑事处罚 |

## 面试要点速记 🎯

1. **单一税率 10%** — 简洁透明，无减低税率
2. **GST-free vs Input-taxed** — 核心区分，面试必考
3. **Netflix Tax (2017)** — 数字经济征税先驱
4. **低价值进口 AUD 1,000 规则** — 消除跨境免税优势
5. **反向征收仅在无完全抵扣权时有效** — 理解实际税负
6. **AUD 75,000 注册门槛** — 国际对比
7. **简化注册** — 非居民专用，无需 ABN
8. **GST 集团 90% 控制** — 与 EU Organschaft 对比
9. **10 年不动产调整** — 与 EU 标准比较
10. **ATO 电子申报强制** — 高度数字化税务管理
