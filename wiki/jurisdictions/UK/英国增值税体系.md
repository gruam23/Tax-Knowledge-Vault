# 英国增值税体系 (United Kingdom — VAT)

## 速览
- 标准税率20%，减低5%，零税率0%（食品/童装/书籍等）
- 脱欧后EU货物视同进口，可选Postponed VAT Accounting延期入账
- Making Tax Digital(MTD)要求所有VAT注册企业数字化申报


> 英国脱欧后不再适用 EU VAT Directive，但保留了类似结构。北爱尔兰通过 Windsor Framework 在货物方面仍适用 EU 规则。

## 一、速览

| 项目 | 内容 |
|------|------|
| **税名** | Value Added Tax (VAT) |
| **引入日期** | 1973年4月1日 |
| **管理机关** | HM Revenue & Customs (HMRC) |
| **标准税率** | **20%** |
| **减低税率** | **5%** |
| **零税率** | **0%** |
| **税号格式** | GB + 9位数字 / XI 前缀（北爱尔兰 EU 交易用） |
| **申报周期** | 季度（可申请月度） |
| **注册门槛** | **GBP 90,000**（12个月滚动） |
| **非居民退税** | 有（通过 Directive 13 程序） |

## 二、税率

| 税率 | 适用范围 |
|------|---------|
| **20%** | 大多数货物和服务 |
| **5%** | 家用燃料/电力、儿童汽车座椅、卫生用品、能源节约材料安装 |
| **0%** | 食品（非餐饮）、儿童服装、书籍报刊、公共交通、处方药、出口、新建住宅 |
| **免税** | 金融/保险、教育、医疗、邮政服务、住宅不动产租赁、墓地火葬 |

## 三、纳税人

### 注册门槛
- **回溯测试**：过去 12 个月应税营业额 > GBP 90,000 → 30 天内注册
- **前瞻测试**：预计未来 30 天内将超过门槛 → 即时注册
- 自愿注册无门槛限制

### VAT 集团注册
- 允许受同一控制的实体组成 VAT 集团
- 集团内交易**不征 VAT**
- 代表成员负责申报，所有成员**连带责任**
- 非 UK 居民实体可加入（需满足条件）

### 非居民企业
- 在 UK 从事应税供应 → 须注册（无门槛豁免）
- **无须**指定税务代理人
- 须提供银行保证金或提供 UK 地址

## 四、脱欧影响 (Brexit) ⭐

### 货物进出口
- UK-EU 货物贸易现按**进出口**规则处理（非共同体内供应/取得）
- 进口 VAT 通过 **postponed VAT accounting** 延期入账（在 VAT 申报表中同时申报销项和进项）
- 出口到 EU → 零税率（需出口证据）

### 北爱尔兰 (Windsor Framework)
- 北爱尔兰在**货物方面**仍适用 EU VAT 规则
- 使用 **XI 前缀**的 VAT 号进行 EU 货物交易
- **服务方面**适用 UK VAT 规则
- 北爱尔兰企业可能需要同时遵守 UK 和 EU 规则

### 跨境服务
- UK 被视为 EU 外的第三国
- EU 向 UK B2B 服务 → UK 企业反向征收
- UK 向 EU B2B 服务 → EU 企业反向征收

## 五、反向征收

- 从海外接收的 B2B 服务 → 买方反向征收
- **国内反向征收**：
  - 建筑和建设服务（CIS 反向征收）
  - 手机和电子产品（≥ GBP 5,000）
  - CO₂ 排放配额
  - 可再生能源证书
  - 天然气和电力（批发交易）

## 六、数字经济

- 非 UK 企业向 UK 消费者提供数字服务 → 须在 UK 注册并缴纳 VAT
- **在线平台**（OMP）促成交易时视为供应商：
  - 非 UK 供应商通过 OMP 向 UK 消费者销售 ≤ GBP 135 的货物
  - OMP 须收取并缴纳 UK VAT

## 七、进项税抵扣

### 不可抵扣
- 商务招待费（向非员工提供的招待）
- 私人用途支出
- 购买/租赁/维护汽车（除非 100% 业务用途或用于出租/驾驶教学）

### 可抵扣
- 差旅住宿、商务差旅
- 燃油（业务用途比例）
- 会议、培训
- 广告

### 按比例抵扣 (Partial Exemption)
- 标准方法：按应税/免税营业额比例分摊
- **微量规则** (de minimis)：每月免税相关进项税 ≤ GBP 625 且 ≤ 总进项税 50% → 全额抵扣

### 资本品调整 (Capital Goods Scheme)
- 电脑及电脑设备 ≥ GBP 50,000 → **5 年**调整
- 土地/建筑物 ≥ GBP 250,000 → **10 年**调整
- 飞机/船舶/其他船只 ≥ GBP 50,000 → **5 年**调整

## 八、申报与缴纳

| 项目 | 规则 |
|------|------|
| **频率** | 季度（可申请月度） |
| **截止** | 申报期结束后 **1个月零7天** |
| **电子申报** | **Making Tax Digital (MTD)** 强制 — 须通过兼容软件提交 |
| **缴纳方式** | 银行转账、Direct Debit |
| **年度申报** | 可选择年度核算方案 (Annual Accounting Scheme) |

### 特殊方案
- **现金制** (Cash Accounting)：营业额 ≤ GBP 1,350,000 可选
- **统一费率方案** (Flat Rate Scheme)：营业额 ≤ GBP 150,000 — 按行业固定比率缴 VAT
- **零售方案** (Retail Schemes)：零售商简化计算

## 九、罚则

| 违规 | 罚则 |
|------|------|
| 迟交申报 | 积分制：积累足够积分后 → GBP 200 罚款 |
| 迟缴税款 | 15天后 2% + 30天后再 2% + 此后 4%/年 |
| 错误 | 合理注意 0%；无合理注意 15%–30%；故意 35%–70%；故意隐瞒 50%–100% |
| 追诉时效 | 一般 **4 年**；故意/不当行为 **20 年** |

## 面试要点速记 🎯

1. **标准税率 20%，零税率适用广泛**（食品、童装、新建住宅）
2. **脱欧后 UK-EU 货物按进出口处理** — postponed VAT accounting
3. **北爱尔兰 Windsor Framework** — 货物适用 EU 规则，服务适用 UK 规则
4. **MTD 强制数字申报** — 必须通过兼容软件
5. **GBP 90,000 注册门槛** — EU 中较高水平
6. **Flat Rate Scheme** — 小企业简化方案，面试常问
7. **Partial exemption de minimis** — GBP 625/月门槛
8. **在线平台 GBP 135 规则** — 低价值进口货物
9. **新积分制罚则体系** — 替代了旧的固定罚款制
10. **XI 前缀** — 北爱尔兰 EU 货物交易专用
