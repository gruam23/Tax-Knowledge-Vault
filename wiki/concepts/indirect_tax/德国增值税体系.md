# 德国增值税体系 (Germany — Umsatzsteuer / USt)

## 速览
- 标准税率19%，减低税率7%（食品/书籍/公共交通等）
- 小企业豁免门槛€22,000，非居民无门槛须直接注册
- 月度预申报+年度终算，电子发票正在推进中


> 德国是欧盟最大经济体，VAT 制度严格遵循 EU VAT Directive，同时在注册、申报和罚则方面有本国特色。

## 一、速览 (At a Glance)

<!-- confidence: EXTRACTED -->

| 项目 | 内容 |
|------|------|
| **税名** | 增值税 (Umsatzsteuer, USt) |
| **引入日期** | 1968年1月1日 |
| **管理机关** | 联邦财政部 (Bundesministerium der Finanzen) |
| **标准税率** | **19%** |
| **减低税率** | **7%** |
| **其他** | 零税率 (0%)、免税 |
| **税号格式** | DE + 9位数字 (如 DE123456789) |
| **申报周期** | 月度/季度 + 年度申报 |
| **注册门槛** | **无**（境内进行应税交易即须注册；小企业豁免见下文） |
| **非居民退税** | 有（EU 企业 / 非 EU 企业均可，程序不同） |

## 二、应税范围 (Scope)

<!-- confidence: EXTRACTED -->

德国 VAT 适用于：
- 纳税人在德国境内实施的有偿**货物供应**
- 纳税人在德国境内实施的有偿**服务供应**
- 货物的**共同体内取得** (Intra-Community Acquisition)
- 货物**进口**

### 持续经营转让 (TOGC)
- 转让全部或部分业务资产时，如受让方继续经营 → **不属于应税供应**
- 转让方不收 VAT，受让方视为转让方的权利义务继承人

### 关联方交易
- 无特殊关联方定价规则
- 但若交易价格**明显偏低/偏高且无商业合理性** → 税务机关可按**公开市场价值**重新确定计税基础

## 三、纳税人 (Who is Liable)

<!-- confidence: EXTRACTED -->

- **纳税人**：在德国境内独立从事经济活动的任何人
- 经济活动包括：商业、工业、专业服务和矿业活动
- 法律形式不限（公司、合伙企业、个人企业）

### 小企业豁免 (Kleinunternehmerregelung)
- 上一年营业额 ≤ **EUR 25,000** 且当年预计 ≤ **EUR 100,000** → 可免征 VAT
- 免税小企业**不得抵扣进项税**
- 自 2025年1月1日起，跨成员国小企业免税规则适用（EU 全域营业额 ≤ EUR 100,000）

### VAT 集团注册 (Organschaft)
- 德国允许 VAT 集团注册，称为 **Organschaft**
- 条件：子公司在财务、组织和经济上**与母公司紧密关联**
- 集团成员间交易**不征 VAT**
- 母公司（Organträger）负责整个集团的 VAT 申报
- 所有成员对 VAT 债务**承担连带责任**

### 非居民企业
- 在德国从事应税交易的非居民企业须注册 VAT
- **无须**指定税务代理人（EU 企业直接注册）
- 非 EU 企业须指定**税务代理人**（Fiskalvertreter），但受限条件较多

## 四、税率 (Rates)

<!-- confidence: EXTRACTED -->

| 税率 | 适用范围 |
|------|---------|
| **19%** 标准税率 | 大多数货物和服务 |
| **7%** 减低税率 | 食品、书籍报刊、公共交通（短途≤50km）、住宿、文化活动门票、农产品、医疗器械、艺术品 |
| **0%** 零税率 | 共同体内供应、出口 |
| **免税** | 银行/保险服务、医疗服务、教育、不动产租赁（住宅）、邮政服务 |

**要点**：
- 餐饮外卖适用 7%，堂食适用 19%
- 不动产租赁原则上免税，但**可选择放弃免税** (Option to Tax)，条件是承租人使用不动产进行应税活动

## 五、反向征收 (Reverse Charge)

<!-- confidence: EXTRACTED -->

德国适用反向征收的场景：
1. **跨境服务**：非居民向德国纳税人提供的 B2B 服务 → 买方反向征收
2. **建筑工程服务**：本国建筑承包商向另一建筑商提供服务
3. **CO₂ 排放配额交易**
4. **手机、平板电脑、游戏机**（供应价值 ≥ EUR 5,000/单笔发票）
5. **金属和有色金属**（供应价值 ≥ EUR 5,000/单笔发票）
6. **天然气、电力、供暖/制冷**（供应给经销商）
7. **电信服务**（向电信经销商）

## 六、数字经济 (Digital Economy)

<!-- confidence: EXTRACTED -->

- B2C 数字服务 → 按消费者所在国税率征税，可通过 **OSS 一站式申报**
- B2B 数字服务 → 客户在其所在国**反向征收**
- **在线平台规则**（2021年7月起）：促成交易的平台可被视为供应商

## 七、进项税抵扣 (Input Tax Recovery)

<!-- confidence: EXTRACTED -->

### 基本规则
- 进项税用于应税交易 → **可抵扣**
- 进项税用于免税交易 → **不可抵扣**
- 须持有符合要求的**VAT 发票**

### 不可抵扣项目
- 50% 的商务招待费
- 用于员工私人消费的支出
- 用于免税交易的进项

### 可抵扣项目（与应税业务相关时）
- 汽车购买/租赁/维护/燃油
- 差旅费、会议费
- 住宿费
- 移动电话
- 商务礼品（≤ EUR 50/人/年）

### 按比例抵扣 (Pro Rata)
- 同时从事应税和免税活动 → 按营业额比例分摊
- 部分行业可申请按**实际使用**方法分摊

### 资本品调整
- 初始抵扣的进项税需在调整期内按实际使用变化进行年度调整
- 动产：**5 年**调整期
- 不动产：**10 年**调整期

## 八、发票 (Invoicing)

<!-- confidence: EXTRACTED -->

### 必要内容
德国 VAT 发票必须包含：
- 供应商和客户的全名和地址
- 供应商的 VAT 号或 EU VAT 识别号
- 开票日期和供应日期
- 顺序编号
- 货物/服务数量和性质描述
- 税前金额、适用税率、VAT 金额
- 反向征收或免税交易须注明依据

### 电子发票
- 允许且日益普及
- 须满足 **GoBD** 标准（电子归档合规要求）
- **XRechnung** 格式用于 B2G 供应
- 须保证发票的**真实性、完整性和可读性**

### 简化发票
- 含税金额 ≤ **EUR 250** → 可使用简化发票

## 九、申报与缴纳 (Returns & Payment)

<!-- confidence: EXTRACTED -->

| 项目 | 规则 |
|------|------|
| **预申报周期** | 上年 VAT 税额 > EUR 9,000 → **月度**；≤ EUR 9,000 → **季度**；≤ EUR 2,000 → 可免预申报 |
| **预申报截止** | 次月/季后第 **10 天** |
| **年度申报截止** | 次年 **7月31日**（聘用税务师可延至次次年 2月底） |
| **缴纳方式** | 银行转账（不接受现金） |
| **电子申报** | **强制**，通过 ELSTER 平台 |
| **新注册企业** | 注册后前两年**必须月度申报** |

### EU 补充申报
- **Intrastat**：发货 ≥ EUR 1M / 收货 ≥ EUR 3M → 月度统计申报
- **EU 销售清单 (ESL)**：货物月度、服务季度，截止日为次月 25 日

## 十、罚则 (Penalties)

<!-- confidence: EXTRACTED -->

| 违规类型 | 罚则 |
|---------|------|
| 迟缴税款 | **1%/月** × 取整至 EUR 50 的欠税额 |
| 迟交申报 | 最高 **10%** 应纳税额，上限 EUR 25,000 |
| 一般错误 | **0.15%/月**利息（自评估日起） |
| 欺诈 | 10%–20% 附加税 + **0.5%/月**利息；刑事处罚最高 **10 年**监禁 |
| 追诉时效 | 一般 **4 年**，欺诈 **5–10 年** |
| 公司高管个人责任 | 仅限欺诈行为，可对相关自然人追责 |

## 十一、特殊制度 (Special Schemes)

<!-- confidence: EXTRACTED -->
- **二手货物保证金计税** (Margin Scheme)：如二手车商从私人处采购后转售
- **旅行社保证金计税** (TOMS)：非 EU 旅行社过渡期延至 2026年12月31日
- **现金制** (Istbesteuerung)：上年营业额 ≤ EUR 800,000 的小企业可选择按收付实现制计 VAT
- **农林特殊税率**：适用专门税率

## 面试要点速记 🎯

> 针对 European Tax Internship 面试的德国 VAT 高频知识点

1. **标准税率 19%，减低税率 7%** — 与其他 EU 成员国对比（如法国 20%/5.5%）
2. **Organschaft** — 德国 VAT 集团注册制度，母公司承担全部申报义务
3. **反向征收**适用范围广 — 不仅限跨境，建筑、金属、电信等本国交易也适用
4. **ELSTER 强制电子申报**，新企业前两年必须月报
5. **不动产免税可选择放弃** (Option to Tax) — 常见面试考点
6. **小企业门槛 EUR 25,000** — 2025 年起与 EU 跨境免税协调
7. **10年不动产调整期** — 与标准 5 年期形成对比
8. **记录保存期 10 年** — 德国在 EU 中属于较长的
9. **罚金 1%/月迟缴利率** — 在 EU 内相对较高
10. **GoBD 电子归档标准** — 电子发票合规的核心要求
