# 中国增值税体系 (China Mainland — VAT / 增值税)

## 速览
- 三档税率：13%标准/9%交通运输建筑农产品/6%现代服务
- 小规模纳税人3%征收率（年销售额≤500万），简易计税
- 金税四期全面推行电子专用发票，进项抵扣凭票控税


> 中国于 2026年1月1日正式施行《增值税法》，将原暂行条例上升为法律。一般纳税人/小规模纳税人双轨制是核心特色。

## 一、速览

| 项目 | 内容 |
|------|------|
| **税名** | 增值税 (Zeng Zhi Shui) |
| **引入日期** | 1994年1月1日（暂行条例）；2026年1月1日（增值税法） |
| **管理机关** | 财政部 (MOF) + 国家税务总局 (STA) |
| **标准税率** | **13%** |
| **减低税率** | **9%、6%、3%、5%** |
| **零税率** | 0%（出口、特定跨境服务） |
| **申报周期** | 10天/15天/月度/季度 |
| **注册门槛** | 年营业额 ≥ CNY 500万 → 一般纳税人 |

## 二、纳税人双轨制 ⭐

| 类别 | 年营业额 | 进项抵扣 | 税率 |
|------|---------|---------|------|
| **一般纳税人** | ≥ CNY 500万 | ✅ 可 | 13%/9%/6% |
| **小规模纳税人** | < CNY 500万 | ❌ 不可 | 3%（减按1%至2027年底） |

- 小规模纳税人可自愿申请转为一般纳税人
- 月销售额 ≤ CNY 10万（季度 CNY 30万）的小规模纳税人**免征 VAT**（至 2027 年底）

## 三、税率

| 税率 | 适用范围 |
|------|---------|
| **13%** | 大多数货物、有形动产租赁 |
| **9%** | 农产品、自来水、天然气、图书报刊、交通运输、建筑服务、不动产 |
| **6%** | 现代服务（研发、信息技术、咨询、文化创意）、金融、生活服务、电信增值 |
| **5%** | 不动产（简易计税）、劳务派遣（简易计税） |
| **3%** | 小规模纳税人、特定货物（简易计税） |
| **0%** | 出口货物、特定跨境服务（研发、设计、软件等） |

## 四、反向征收（代扣代缴）

- 中国**不使用反向征收**术语，而是**代扣代缴机制**
- 境外企业在中国境内发生应税行为 → **购买方为扣缴义务人**
- 扣缴义务人可将代扣 VAT 作为进项税抵扣

## 五、数字经济与跨境电商

### 跨境电商零售进口
- 单次限额 **CNY 5,000**，年度限额 **CNY 26,000**
- 限额内：关税 0%，进口 VAT 和消费税按法定税率 **70%** 征收
- 超限额：全额征税

### 平台经济
- 2025年6月起生效的**互联网平台企业涉税信息报告规定**
- 平台须向税务机关报送第三方卖家信息
- 境外平台也须报告在中国提供营利性服务的信息

## 六、进项税抵扣

### 基本规则
- 一般纳税人凭**增值税专用发票**抵扣
- 海关进口 VAT 完税凭证可抵扣
- 2017年1月1日后开具的发票**无认证时限**

### 不可抵扣项目
- 用于简易计税项目的进项
- 用于免税项目的进项
- 非正常损失相关进项
- 集体福利和个人消费
- **贷款服务、餐饮、居民日常服务、娱乐服务**（直接消费）

### 2026年新法变化
- **年度进项税对账**：纳税人须在次年1月申报期内完成
- **非应税交易**进项不可抵扣（如股息、汇兑收益）
- **高价值混用资产**（> CNY 500万）：取得时全额抵扣 → 使用年限内年度调整

### 留抵退税制度 ⭐
- 增值税法首次将留抵退税上升为法定权利
- 制造业、科技等行业：**月度**申请退税，比例 100%
- 房地产业：连续 6 个月增量 ≥ CNY 50万，退 60%
- 其他行业：阶梯式退税（≤ CNY 1亿退 60%，超 CNY 1亿退 30%）

## 七、发票

- **增值税专用发票**：一般纳税人之间使用，是进项抵扣凭证
- **增值税普通发票**：向消费者或小规模纳税人开具
- 全面推行**电子发票**（全电发票/数电发票）
- 红字发票（credit note）用于退货/折让调整

## 八、申报与缴纳

| 项目 | 规则 |
|------|------|
| **一般纳税人** | **月度**申报，次月 15 日前 |
| **小规模纳税人** | 可选**季度**申报 |
| **电子申报** | 广泛使用，通过电子税务局 |
| **代扣代缴** | 扣缴义务人在扣缴日起 15 日内申报缴纳 |

## 九、罚则

| 违规 | 罚则 |
|------|------|
| 迟缴税款 | 每日 **0.05%** 滞纳金（约 18.25%/年） |
| 偷税 | 50%–500% 罚款 + 追缴税款 + 滞纳金 |
| 虚开增值税专用发票 | 刑事处罚，最高**无期徒刑** |
| 追诉时效 | 一般 **3 年**，特殊 **5 年**，无限期（偷税） |

## 面试要点速记 🎯

1. **2026年增值税法**正式施行 — 暂行条例升法
2. **双轨制**：一般纳税人（抵扣制）vs 小规模纳税人（简易征收）
3. **13%/9%/6% 三档主要税率** — 与 EU 单税率体系对比
4. **代扣代缴**替代反向征收 — 购买方为扣缴义务人
5. **留抵退税**首次法定化 — 解决企业现金流问题
6. **跨境电商 CNY 5,000/26,000 限额** — 70% 税率优惠
7. **增值税专用发票是抵扣核心凭证** — 与 EU 发票制度对比
8. **全电发票**推广 — 中国数字化税务走在全球前列
9. **虚开发票刑事处罚**极为严厉 — 最高无期徒刑
10. **年度进项对账**2026 年新要求 — 纳税人主动履行义务
